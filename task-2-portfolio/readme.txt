## 🌐 Task 2 – Student Digital Portfolio

For Task 2, I set up my digital presence across three platforms: **GitHub, LinkedIn, and Kaggle**. Each platform serves a distinct purpose in the academic and professional world.

---

### 💻 Platforms Used

- **🐙 GitHub**  
  GitHub is a platform for version control and collaborative software development. I created a profile README repository that includes my name, branch, year, and a sentence about my learning goals. This repository will serve as a living portfolio of my coding projects throughout my B.Tech degree.

- **🔗 LinkedIn**  
  LinkedIn is the world's largest professional networking platform, used by recruiters, companies, and students alike. I filled in my Education section with my current degree, institution (VIT Bhopal University), and expected graduation year.

- **📊 Kaggle**  
  Kaggle is a data science and machine learning community where students can practice real-world datasets and participate in competitions. I created a basic profile to begin exploring data-driven problem-solving.

---

### 🚀 Future Plans

Over the next four years, I plan to use GitHub to host all my coding assignments and personal projects, LinkedIn to connect with professionals and apply for internships, and Kaggle to sharpen my data analysis skills. Setting up these profiles early has made me realize how important a digital presence is, even at the start of an engineering career.
